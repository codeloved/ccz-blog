## git branch
```bash
# 列出所有的分支
git branch 
# 列出所有分支的最后一次提交
git branch -v
# 列出已经合并到当前分支的分支
git branch --merged
# 列出没有合并奥当前分支的分支
git branch --no-merged
# 删除分支
git branch -d 分支名称
```
